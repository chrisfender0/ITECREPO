New York, September 12

Pro-Athlete™ - Fighting Childhood Obesity New game pushes kids to be Pro-Athletes

Pro-Athlete™ a new game released by Plush Packet is raising eyebrows. In an effort to fight childhood obesity, the game encourages children to strive to be pro-athletes. The company recently released the game to a small market in cities like New York, Baltimore and Atlanta.
